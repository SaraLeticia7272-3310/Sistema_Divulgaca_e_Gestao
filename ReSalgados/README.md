# 🥟 Re Salgados — Sistema Web de Divulgação e Gestão

> Plataforma web desenvolvida para a **Re Salgados**, com o objetivo de divulgar produtos, fortalecer a presença digital da marca e oferecer um painel administrativo para gerenciamento de conteúdo.

---

## 📖 Sobre o Projeto

O **Re Salgados** é um sistema web desenvolvido como um projeto de estudo e portfólio, inspirado em necessidades reais de um pequeno negócio.

O projeto busca oferecer uma experiência moderna tanto para os clientes quanto para o administrador da empresa, permitindo divulgar produtos, gerenciar informações e facilitar o contato via WhatsApp.

Além do desenvolvimento da aplicação, o projeto também segue uma documentação completa de arquitetura, UX, banco de dados e boas práticas de engenharia de software.

# 🏗️ Arquitetura

O projeto utiliza uma arquitetura baseada em **MVC Estendido**, separando responsabilidades entre:

* Controllers
* Services
* Repositories
* Models
* Middleware

Essa organização facilita manutenção, reutilização de código e evolução do sistema.
